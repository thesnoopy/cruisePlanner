# Cruise Planner Support

Cruise Planner is an iPhone app for private cruise planning and offline access to your trip information.

If you need help, want to report a problem, or would like to share feedback, please contact:

**Email:** cruisecompanionsupport@mailsmart.de

When contacting support, please include:
- app version
- iPhone model
- iOS version
- a short description of the issue

Cruise Planner helps organize itinerary details, excursions, travel bookings, and important documents in one place.

WebDAV sync is optional. The app can also be used without WebDAV.
